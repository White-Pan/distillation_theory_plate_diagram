# 化工精馏理论塔板绘制
## 1. 简介
以下代码可计算并绘制化工原理中精馏理论塔板的绘制。输入理想状态下平衡线参数α（**仅理想平衡线**）、进料热状态q、实际回流比R、轻组分进料摩尔组成z_F、轻组分塔顶摩尔组成x_D、轻组分塔底摩尔组成x_W即可绘制全回流以及部分回流的塔板图（包括最佳进料板）。
  
可用于完成化工课程设计的部分内容。可以先掌握OriginLab的绘制方法加深化工原理的理解有利于考试。
## 2. 使用方法
要求**PySide6，numpy，matplotlib**库  
运行main.py即可
