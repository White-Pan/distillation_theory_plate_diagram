# 化工精馏理论塔板绘制
## 一、简介
以下代码可计算并绘制化工原理中精馏理论塔板的绘制。输入理想状态下平衡线参数α（**非理想物系的说明见后文**）、进料热状态q、实际回流比R、轻组分进料摩尔组成z_F、轻组分塔顶摩尔组成x_D、轻组分塔底摩尔组成x_W即可绘制全回流以及部分回流的塔板图（包括最佳进料板）。
  
可用于完成化工课程设计或化工原理实验的部分内容。可以先掌握OriginLab的绘制方法加深化工原理的理解有利于考试。
## 二、使用方法
### 1. 运行 
#### 1.1 通过源代码运行
要求**PySide6，numpy，matplotlib，pandas，openpyxl，scipy**库。  
运行main.py即可
#### 1.2 通过exe运行  
直接点击右边的releases并下载exe文件, 等待5~10秒等待程序打开
### 2. 使用
(1) 选择理想/非理想物系  
(2) 输入α（**非理想物系无需输入，但必须导入非理想物系的相平衡数据，见后文第3条**）、q(全回流无需输入)、R(全回流无需输入)、z_F(全回流无需输入)、x_D、x_W  
(3) 选择选择是计算全回流还是部分回流  
(4) 点击“计算并作图”
### 3.非理想物系
相平衡数据示例可见文件夹 "相平衡数据示例 > 乙醇-水相平衡数据.xlsx(或乙醇-水相平衡数据.csv)"  
对于非理想物系的使用方法有特殊要求：必须导入非理想物系的相平衡数据, 相平衡数据表格可用Excel制作。详细的要求可点击应用程序内的"文件导入帮助..."按钮
